# Frontend React + Vite

Modern React frontend using Vite for customer churn prediction.

## Setup

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Build for production
npm run build
```

## Features

- ⚡ Vite for fast development
- ⚛️ React 18 with Hooks
- 🎨 Tailwind CSS styling
- 📊 Interactive prediction form
- 🔌 API integration with FastAPI backend
- 📱 Fully responsive design

## Structure

```
src/
├── main.jsx           # Entry point
├── App.jsx            # Main component
├── index.css          # Global styles
└── components/
    ├── ChurnForm.jsx  # Input form
    ├── ResultCard.jsx # Prediction results
    └── APIStatus.jsx  # API status indicator
```

## Development

Frontend runs on **http://localhost:5173**

Make sure the API is running on **http://localhost:8000**
